# 프리랜서 해피콜 및 감사(Audit) 시스템

## 프로젝트 개요
프리랜서가 고객에게 해피콜을 진행하고, 녹취 파일을 업로드하며, 정해진 체크리스트를 제출하는 시스템입니다. 관리자는 비정상 건을 검토하고 처리합니다.

## 기술 스택
- **백엔드:** Python (Flask)
- **데이터베이스:** SQLite (SQLAlchemy ORM)
- **프론트엔드:** HTML, Tailwind CSS (모바일 반응형)

## 프로젝트 구조
```
/
├── app.py                 # 메인 Flask 애플리케이션
├── templates/
│   ├── base.html          # 기본 레이아웃 템플릿
│   ├── login.html         # 로그인 페이지
│   ├── freelancer_dashboard.html  # 프리랜서 대시보드
│   ├── customer_detail.html       # 고객 상세/체크리스트
│   ├── admin_dashboard.html       # 관리자 대시보드
│   └── admin_submission_detail.html  # 제출 건 상세
├── uploads/               # 녹취 파일 저장 폴더
└── happycall.db          # SQLite 데이터베이스
```

## 주요 기능
1. **사용자 인증:** 프리랜서/관리자 역할 구분
2. **프리랜서 대시보드:** 할당된 고객 리스트 표시
3. **해피콜 프로세스:** 서류 확인 → 스크립트 → 녹취 업로드 → QC 체크리스트
4. **관리자 대시보드:** 제출 건 관리, 비정상 필터링, 처리완료 변경

## 테스트 계정
- **관리자:** 아이디 `1` / 비밀번호 `1`
- **프리랜서:** 아이디 `2` / 비밀번호 `2`

## QC 체크리스트 항목
1. 할부금 안내
2. 위약금 안내
3. 요금제 안내
4. 유지기간 안내
5. 요금 안내
6. 중고폰 반납 안내

**비정상 판정 로직:** 6개 항목 중 하나라도 비정상이면 최종 결과가 '비정상'으로 처리됩니다.

## Mock 데이터 설명
프리랜서 로그인 시 `generate_mock_customers()` 함수가 가상 고객 데이터를 생성합니다. 추후 외부 API 연동 시 이 함수를 API 호출로 교체하면 됩니다.

## 개발 환경
- 서버: `python app.py` (포트 5000)
- 디버그 모드 활성화
